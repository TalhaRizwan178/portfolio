Featured Project

SkillSphere

An AI-Powered Personalized Education & Learning Platform

A full-featured LMS with an integrated AI tutor, auto-generated flashcards, quizzes, lecture packages, and PDF certificate generation — deployed live on cPanel and serving real users.

  • Multi-role RBAC (Student, Expert, Instructor, Admin) with granular permission control.
  • Deployed on cPanel with Apache mod_rewrite proxying to Node.js backend; 33+ Sequelize models.
  • Integrated OpenAI API for AI tutoring, text-to-speech, speech-to-text, and smart content generation.
  • Built with React Native for cross-platform support and seamless mobile experience.
  • Hand-raise interactivity: Students can raise their hand mid-lecture — the AI tutor pauses, listens to the question via speech-to-text, answers it, then seamlessly resumes the lecture from where it left off.
  • Intelligent lecture generation: The AI dynamically determines during lecture creation where to render an image, whiteboard, flowchart, diagram, or code block — choosing the right visual format for each concept automatically.
